
# Covid App
Covid app that shows covid cases in Poland. Made it to train tailwind css, typescript and API



## Run Locally

Clone the project

```bash
  git clone https://github.com/Butterski/covid-app
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
